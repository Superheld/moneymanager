Proportional ring for allocation (Anlagen) and category spend (Analysen).

```jsx
<Donut size={160} stroke={20} center="16k" sub="6 Mt"
  segments={[{value:8034,color:'var(--ink-2)',label:'Wohnen'}, {value:3120,color:'var(--accent)',label:'Lebensmittel'}]} />
```

Always accompany with a dot legend. Use muted neutrals + one teal accent segment; avoid rainbow palettes.
